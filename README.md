  # php_core_crud
  PHP Core CRUD Application
  
  This project is developed using Core PHP and MySQL. It demonstrates CRUD (Create, Read, Update, Delete) operations by connecting a web form with a database. Users can add, view, update, and delete records through a simple and user-friendly interface.
  
  Features:
  
  Database connection using PHP and MySQL
  Insert data through forms
  Display records from database
  Update existing records
  Delete records
  Form validation
  Clean and structured code
  
  Technologies Used:
  
  PHP (Core PHP)
  MySQL
  HTML
  CSS
  bootstrap
  
  Project Objective:
  The main objective of this project is to understand database connectivity and implement CRUD operations using Core PHP and MySQL.
